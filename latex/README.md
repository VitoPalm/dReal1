# LaTeX report

This folder is a self-contained LaTeX build of `../rrrrrreporrrrrrt.md`.

## Build

```sh
make
```

The generated PDF is `report.pdf`.

## Refresh from markdown

```sh
make content
```

`content.tex` is generated by `tools/md_to_tex.py`, which is intentionally scoped
to the markdown style used in this project report.

## Figures

The current plot images were copied into `figures/plottwists/` so the folder can
compile without depending on paths outside `latex/`.
